🤖 ทฤษฎีของอัลกอริทึม

  • Logistic Regression: สำหรับ Classification แบบ Binary
  • Decision Tree: แยกข้อมูลตามคุณสมบัติ เพื่อหาความสัมพันธ์เชิง Non-linear
  • Random Forest: รวมหลาย Tree เพื่อลด Overfitting และเพิ่ม Accuracy
  • Ensemble Learning: การรวมผลลัพธ์หลายโมเดล เช่น Voting/Averaging เพื่อความเสถียร

⚙️ ขั้นตอนการพัฒนาโมเดล

  1. แบ่งข้อมูลเป็น Train/Test
  2. Train โมเดลด้วย Training Set
  3. ประเมินผลด้วย Test Set และ Cross Validation
  4. ปรับ Hyperparameters เพื่อเพิ่ม Accuracy
