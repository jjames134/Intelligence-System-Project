🤖 ทฤษฎีของอัลกอริทึม

  • Deep Neural Network: มีหลาย Hidden Layers เพื่อเรียนรู้ Feature ที่ซับซ้อน
  • Activation Function: ReLU สำหรับ Hidden Layer, Softmax สำหรับ Output
  • Loss Function: ใช้ Cross-Entropy สำหรับ Classification
  • Optimizer: เช่น Adam, SGD เพื่อปรับ Weight ของ Network

⚙️ ขั้นตอนการพัฒนาโมเดล

  1. Clean และ Encode ข้อมูล
  2. Normalize/Scale Features
  3. กำหนด Architecture ของ DNN
  4. Train โมเดลด้วย Training Set และ Validate ด้วย Validation Set
  5. ประเมินผลด้วย Accuracy, Precision, Recall, F1-score, ROC-AUC
  6. ปรับ Hyperparameters (Learning Rate, Layer Size, Epoch, Batch Size)
